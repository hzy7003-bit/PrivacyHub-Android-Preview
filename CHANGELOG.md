# 更新日志

## v0.9.25-beta

新增：

- Pro 系统自动填充服务，可由用户主动选择本地保存的账号资料并填入标准登录表单。
- Pro 网盘提取码辅助，可在用户主动开启无障碍服务后向浏览器页面填入提取码。
- Pro 加密离线备份与跨设备恢复流程。

优化：

- 完善百度网盘、夸克网盘和迅雷云盘链接识别及浏览器优先打开流程。
- 网盘提取码辅助仅执行一次性文本填入，不自动点击登录、下载、确认或支付按钮。
- Release APK 启用 R8 代码混淆、压缩和资源收缩。

修复：

- 安全箱详情页删除内容前增加二次确认，降低误删风险。

## v0.9.4-beta

新增：

- 支持淘宝闪购代付口令或链接识别，保存后可通过“去淘宝闪购代付”打开淘宝官方代付页。

优化：

- App 内说明改为置顶的新手指引，通知栏保存作为推荐入口。
- 补充通知权限、通知常驻兼容、快捷设置磁贴和逐 App 剪切板管理说明。

隐私说明：

- 不会自动把淘宝闪购代付内容写入系统剪切板，不会代替用户确认订单或付款。

## v0.9.1-beta

新增：

- 首页新增“最近保存”入口，位于收藏夹下方、常用标签上方。
- “最近保存”进入独立列表，按保存时间查看最近内容。

优化：

- APK 文件名自动包含版本号和构建类型。
- 版本号改为可追踪配置，发布时更容易区分新旧包。

## v0.9.0-beta

新增：

- 分享链接中转
- 本地加密安全箱
- 购物链接识别
- 网盘链接识别
- 链接净化
- 通知栏保存
- 控制中心磁贴保存
- 本地诊断报告
- 深色模式和浅色模式
- 首页常用标签

优化：

- Release 包启用 R8 混淆
- Release 包使用独立签名
- 默认无网络权限

已知问题：

- 部分 ROM 可能杀掉常驻通知
- 部分 App 或输入框不显示“隐私中转站”文本处理入口
- 控制中心磁贴在部分 ROM 上可能无法读取剪切板
